# Snipcart (snipcart)

Snipcart is a developer-first shopping cart platform that enables merchants to add fully functional e-commerce to any website by embedding a JavaScript snippet. It provides a REST API for managing orders, customers, products, discounts, notifications, abandoned carts, and custom shipping methods. The platform is stack-agnostic and integrates with static site generators, headless CMSs, and modern JavaScript frameworks including React, Next.js, Gatsby, Hugo, and Jekyll.

## APIs

### Snipcart REST API

The Snipcart REST API provides programmatic access to your store's data including orders, customers, products, discounts, notifications, abandoned carts, domains, refunds, user sessions, and custom shipping methods. Authentication uses HTTP Basic Auth with a secret API key. Base URL: `https://app.snipcart.com/api`

- Documentation: https://docs.snipcart.com/v3/api-reference/introduction
- Authentication: https://docs.snipcart.com/v3/api-reference/authentication
